 ### Not listed
 
 As with most guitar pedal BOMs, the following components are not listed above:
 
 * Footswitch
 * DC Jack
 * 2x 1/4" audio jack
 * Enclosure: this pedal fits into a 125B enclosure.
 * Sockets (for the ICs).
 * Wire
 * Solder
